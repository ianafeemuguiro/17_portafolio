Nuestro Portafolio

Conociendo HTML

En este primer ejercicio aprendimos a crear un documento html, por primera vez nos acercamos al uso de las etiquetas y como nos ayudan a distribuir el contenido.

Lenguajes de la Web

Aprendimos a utilizar el CSS para darle estilo a nuestro contenido y dejar de utilizar estilo en nuestro HTML. También aplicamos más sobre la teoría en los ejercicios.

Nuestras Coders

En este ejercicio aprendimos el uso de cajas para una mejor distribución del contenido y así utilizar div para tener todos los elementos por bloques.

Chicama

El ejercicio de Chicama nos ayudo a visualizar más un sitio web y como el contenido está conformado por cajas, una grid y rows. Desde este ejercicio en adelante utilizamos más la grid.
